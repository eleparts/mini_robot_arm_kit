버전별 기능과 프로그램 유의사항


1. 스위치 제어의 동작 방향을 바꾸고 싶은 경우

예) D 스위치와 - B 스위치가 각각 손을 오므리고/벌리는 동작을 할 경우, 반대로 동작하도록(벌리고/오므리게) 바꾸고 싶다면 

방법 1. 스위치 모듈이나 쉴드 보드에 연결된 두 선의 한쪽을 서로 바꾸어 줍니다.
방법 2. 프로그램 코드에 있는 핀 번호를 수정한 뒤 다시 write해 줍니다 
      const int pin_Sw2_X  = 7;    // Sw D pin 번호
      const int pin_Sw2_Y  = 5;    // Sw B pin 번호  
      해당 부분의 숫자 7 과 5 를 서로 바꾸어 줍니다.

Arm_kit_v1.31 : 기본 제어기능 및 평행이동 기능 및 기억-실행 기능이 추가되어 있습니다.

      기억 실행 기능 : F 버튼을 눌러 현재 동작을 기억시킵니다. 해당 동작을 최대 [SAVE_DATA_MAX_NUMBER] 횟수만큼 (기본값 30)저장할수 있으며, E 키를 누르면 지금까지 저장된 동작을 실행합니다.
      이때 E 카 (실행 기능)은 누를때마다 1회 실행되며 중간에 취소나 추가 조작이 불가능합니다.

      그 후, 다시 F 키를 누를시 저장된 내역이 초기화 되며 첫번째 동작부터 다시 저장됩니다. (이어서 저장하기 불가능.)


Arm_kit_v1.32 : 기억- 실행 기능의 동작 속도를 조절하는 함수가 추가되었습니다.
